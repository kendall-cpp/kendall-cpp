## 安装fashDFS
```
unzip libfastcommon-1.36.zip

tar zxf fastdfs-5.10.tar.gz 
```

### 安装基础库包
```
cd libfastcommon-master/
./make.sh
sudo ./make.sh install
```
具体见文档吧
